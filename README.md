## 🚀 Como Executar os Testes

### 🔹 API Testing

1. Abrir o Postman
2. Importar a collection:
   `/api-tests/collection/restful-booker.json`
3. Configurar as variáveis de ambiente:
   - base_url
   - token
4. Executar as requisições na ordem:
   - Autenticação
   - CRUD
   - Validações
